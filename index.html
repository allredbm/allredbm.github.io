My Projects

Kohler Scraper

This project is a web scraper that I was asked to write for Specbooks.com. When run, the scraper is designed to iterate through the Kohler.com product catalog and scrape the product information and any documentation for each product. This information is then converted into a csv document and exported, so that it can be used for other purposes.

Github repo: Kohler Scraper

HTML Quiz

This page is an interactive quiz on HTML5's features. The site keeps track of which answers are right and wrong and gives you a total score after the last question is submitted.

Github repo: HTML_Quiz

Stickys

This project is a webpage where a user can create and edit sticky notes that will remain on the page. These notes are stored on an in-browser database and will remain when the page is refreshed.

Github repo: Stickys

Snake

This project is a version of the Snake game coded into the frontend of the webpage. The snake can be controlled with the arrow keys.

Github repo: Snake

Photo Gallery

This project is an animated photo gallery. It uses javascript to sort the photos by tags and uses Fancybox to display the image on click.

Github repo: Photo_Gallery

The TechEdge

The TechEdge is a frontend mockup for a tech blog.

Github repo: TechEdge

Weather App

This weather app was an exercise for Tech Talent South meant to teach API integration in a rails app. The website uses the WUnderground API to take a city as an input and display the current weather in that city. The site also includes a link to a more detailed forecast and a background that changes based on the weather being displayed.

Github repo: tts-weather-app

Good Coffee Co.

Good Coffee Co. was an exercise for Tech Talent South. It is a mockup website for a fictional coffee shop and is designed to advertise for the shop and communicate the shop's values.

Github repo: tts-good-coffee

TTS-Travel

This was a project for Tech Talent South. It is a rudimentary site that uses the Google Maps API. The project was meant to teach API usage, as well as how to make custom icons for Google Maps.

Github repo: tts-travel

Tweeter

This was a project for Tech Talent South. The intent was to design a website that replicated the features of Twitter. Users create an account with Devise and can then create short messages that are posted to a communal wall.

Github repo: tts-tweeter
